AI Shorts Uploader

Automated uploader that posts short videos to the VR-Shorts YouTube channel using the YouTube Data API.

About

This application is used to automatically upload videos processed by an internal pipeline to the channel VR-Shorts. It uses OAuth 2.0 credentials (user consent) and the YouTube Data API v3. No user data is collected beyond standard Google OAuth consent.
